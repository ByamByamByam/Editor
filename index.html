사용 설명서

프로그램을 사용해주셔서 감사합니다.

이 프로그램은 로그 편집 및 연출 설정, 보관하기 위한 리플레이 에디터입니다.

Roll20, 코코포리아, 카카오톡, 메모장 로그를 하나의 프로젝트로 관리하고

배경, BGM, 효과음, 연출을 이용하여 로그를 제작할 수 있습니다.

처음 시작하기

1. 로그 가져오기 또는 직접 채팅을 추가합니다.

2. 채팅 추가시 챕터는 자동생성됩니다.

3. 등장인물을 등록합니다.

4. 필요시 편집모드에서 구간별 배경/BGM/효과음을 추가합니다.

5. 리플레이버튼으로 감상하거나 프로젝트를 저장 또는 HTML로 출력합니다.

6. 저장은 반드시 새 폴더 안에 해주세요.

로그 가져오기

지원 형식

  • Roll20 HTML
  • 코코포리아 HTML
  • 카카오톡 TXT
  • 메모장 TXT

가져오기 시

  • 등장인물 자동 생성
  • 판정 자동 인식 및 CoC 판정 카드 자동 생성
  • (일부 인식이 안될 수 있습니다)
  • 메모장은 [날짜] 이름: 대사 형식을 인식합니다.
  • /desc, /emas 자동 변환
  • (em은 emas로 변환됩니다. 불러온 후 편집시 /em으로 변경하세요.)
  • *로그 내 이미지, css 인식
  • 예: [빨간 글씨](#" style="color:red;")

배경, BGM, 효과음

  • 배경 기본 색
  • (이미지 전환시 나타나는 색이 됩니다)
  • 구간 별 배경 이미지
  • 구간 별 BGM, 효과음
  • html 출력시 기본 배경색상만 지원됩니다.

연출

  • 클릭 액션
  • 메시지 간격
  • 연속 대화 묶기
  • desc, em, emas, as 스타일
  • 웹폰트

리플레이

  • 내가 설정한 연출을 감상할 수 있는 시스템입니다.
  • 연출탭에서 클릭액션을 설정하면 더 몰입할 수 있습니다.
  • Shift를 이용하여 여러 메시지를 선택한 뒤 PNG로 저장.

출력

  • HTML 로그 출력
  • 현재 챕터, 채팅방 전체 단위 html 출력

출력된 HTML은 에디터 없이 브라우저에서 바로 열람할 수 있습니다.

  • 단 BGM, 효과음, 배경이미지, 클릭액션은 지원하지 않습니다.

프로젝트

프로젝트에는 다음 정보가 모두 저장됩니다.

  • 채팅, 챕터 이름 설정
  • 챕터 분리, 챕터 합치기 (선택한 챕터 또는 메세지를 기준으로 포함하여 설정됩니다.)

예) A, B, C, D >> B챕터 선택 후 다음 챕터와 합치기 >> A, BC, D

ABC, D >> B 메시지 선택 후 챕터 분리하기 >> A, BC, D

TIP

  • Ctrl + S : 프로젝트 저장
  • Enter : 메시지 저장
  • Shift + Enter : 줄바꿈
  • Esc : 메시지 선택 해제, 집중모드 돌아가기
  • Shift + 클릭 : 메시지 범위 선택

등장 방식

클릭 액션을 켜면 리플레이시 모든 메시지가 숨겨지고 클릭할 때마다 하나씩 나타납니다.
